Mastermind Canvas

Classic "MasterMind"

Default settings of the canvas allow for a standard Mastermind game with only minimal JS.

Except for the rendering of the "rating" of a given guess, pure HTML and CSS has been used for pretty much everything, from graphic elements to showing, hiding and highlighting elements.
